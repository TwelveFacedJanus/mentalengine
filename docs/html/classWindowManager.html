Detailed Description

template<typename T = GLFWwindow>
class WindowManager< T >

Template class for managing application windows and OpenGL context.

The WindowManager class is responsible for creating and managing the main application window, initializing OpenGL context, and coordinating between the renderer and user interface systems. It provides a clean abstraction over GLFW window management.

Template Parameters
TWindow type (defaults to GLFWwindow)

Key responsibilities:

  • GLFW library initialization and cleanup
  • Window creation and management
  • OpenGL context setup
  • Renderer and UI system coordination
  • Main application loop execution
Note
This class is non-copyable and non-movable for safety
